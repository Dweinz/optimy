# Optimy — Pirate Browser Games

Two complete browser games built with Vite + TypeScript + Three.js. No backend — saves live in localStorage.

## ⚓ Seven Seas Idle (repo root)

A deep idle/incremental pirate game: 13 skills with mastery, 12 activities, fleets, crews, port building, treasure maps, relic collections, bounty bosses, random events, achievements and a prestige layer — with a cozy low-poly 3D harbor that grows as you do.

```bash
npm install
npm run dev
```

## 🏴‍☠️ Pirate Logistics (`pirate-logistics/`)

A Factorio-inspired real-time logistics sim in a procedurally generated archipelago: production chains, shipping routes as conveyor belts, fog-of-war exploration, colonization, a tech tree, markets and smuggling — with live production/consumption stats and a bottleneck report.

```bash
cd pirate-logistics
npm install
npm run dev
```

Both games: `npm run build` produces a static `dist/` deployable to any free static host (Netlify, Surge, GitHub Pages, itch.io).
