# Optimy — Game Design Reference

> **Purpose:** A complete design document for both games in this repo. Hand this to any future session as context so no re-reading of source is needed.

---

## Repository Layout

```
optimy/
├── src/                      ← Seven Seas Idle (idle/incremental)
├── pirate-logistics/src/     ← Pirate Logistics (real-time logistics sim)
├── landing/index.html        ← Landing page linking both games
└── DESIGN.md                 ← this file
```

**Tech stack (both games):** Vite + TypeScript + Three.js (low-poly 3D), localStorage saves, no backend. `npm run dev` / `npm run build` in each directory.

## Game 1 — Seven Seas Idle

### Elevator Pitch
A deep idle/incremental pirate game. One activity runs at a time; everything else ticks in the background. The player grows from a leaky raft with 10 gold into a legendary admiral who can prestige to climb again, stronger.

### Core Loop
```
Pick activity → earn resources & XP → buy buildings / ships / crew
→ unlock harder activities → explore islands → collect relics / defeat bosses
→ reach prestige thresholds → reset with Legend Points
```

### Resources (13 total)
| Resource | Role |
|---|---|
| Gold | Primary currency — everything costs it |
| Crew | Number of crew members; caps at fleet capacity |
| Supplies | Activity fuel; consumed by Explore, passively by fleet |
| Rum | Crew morale booster; brewed in Tavern |
| Maps (4 tiers) | Common/Rare/Epic/Legendary — consumed to discover islands and hunt treasure |
| Treasure | Mid-tier currency; fenced for gold |
| Trade Goods | Generated by Trade/Raid; sold at port |
| Reputation | Mid-game progression gate; needed for prestige |
| Influence | Late-game progression gate; needed for prestige |
| Ships | Count of fleet vessels (derived) |
| Naval Power | Derived combat score (fleet power × crew efficiency) |
| Knowledge | Consumed to study relics; gained passively |
| Cursed Relics | Spent to unlock/slot relic items |

### Skills (13 total, each with XP + mastery)
Every skill has 1–2 `BonusKey` effects that scale per level as a percentage multiplier. Mastery persists through prestige.

| Skill | Key Effect |
|---|---|
| Sailing | Speed +1%/lvl, Supplies +0.5%/lvl |
| Navigation | Speed +0.5%, Treasure +1%/lvl |
| Plundering | Combat +1%, Gold +0.3%/lvl |
| Treasure Hunting | Treasure +1.5%/lvl |
| Trading | Trade +1.5%/lvl; unlocks trade routes at 1/8 levels |
| Smuggling | Smuggle Safety +1%, Gold +0.2%/lvl |
| Shipbuilding | Build Speed +2%, Ship Discount +0.5%/lvl |
| Crew Management | Crew Efficiency +1%, Crew Safety +1%/lvl |
| Diplomacy | Influence +1.5%, Reputation +1%/lvl |
| Cartography | Map Find +1.5%, Treasure +0.5%/lvl |
| Tavern Keeping | Rum +2%, Recruit +1%/lvl |
| Naval Combat | Combat +1.5%, Crew Safety +0.5%/lvl |
| Cursed Lore | Relic Find +1.5%, Knowledge +1%/lvl |

### Activities (12 total, one active at a time)
| Activity | Skill trained | Output |
|---|---|---|
| Sail | Sailing | Gold, supplies, voyage completions (bonus gold) |
| Explore | Navigation | Explore progress → island discoveries |
| Raid | Plundering | Gold, trade goods, reputation (vs. escalating targets) |
| Trade | Trading | Gold, trade goods |
| Smuggle | Smuggling | High gold; risk of bust (fine + crew loss) |
| Treasure Hunt | Treasure Hunting | Treasure; consumes maps |
| Recruit Crew | Crew Management | Hires crew every ~15 s; costs gold |
| Build Ships | Shipbuilding | Advances build order progress |
| Manage Tavern | Tavern Keeping | Rum; random rumor events (maps, crew, gold) |
| Research Maps | Cartography | Knowledge; produces maps on timer |
| Study Relics | Cursed Lore | Knowledge + cursed relics; relic unlock attempts |
| Diplomacy | Diplomacy | Influence, reputation |

All activities are **dt-robust** — rates work correctly for large offline time chunks (up to 12 h simulated).

### Fleet (7 ship tiers)
Raft → Sloop → Brig → Brigantine → Frigate → Man O' War → Legendary Ship

Ships contribute: **Speed** (activity rate multiplier), **Power** (raid/boss combat), **Cargo** (trade rate), **Capacity** (crew slots). Ships require Shipbuilding skill level to unlock; built via `buildOrder` (points system). Can be scrapped for partial refund.

### Crew
Each crew member has: name, role (Deckhand→Sailor→Officer→Quartermaster), level, XP, morale (0–100), combat, navigation, loyalty. `crewEfficiency` is a multiplier on all activity output derived from crew count, morale, and role bonuses. Crew are recruited (costs gold), promoted, and occasionally lost to events/raids.

### Buildings (Port, 9 types)
Built at the home port; scale by level with exponential cost (`baseCost × mult^level`). Each grants a passive `BonusKey` bonus per level.

| Building | Bonus |
|---|---|
| Dock | Speed +2%/lvl |
| Warehouse | Supplies +3%/lvl |
| Tavern | Recruit +3%/lvl, passive rum |
| Shipyard | Build Speed +5%/lvl, passive construction |
| Market | Trade +3%/lvl, auto-sells trade goods |
| Smuggler Den | Smuggle Safety +3%/lvl |
| Map Archive | Map Find +4%/lvl |
| Fortress | Combat +4%/lvl |
| Relic Vault | Relic Find +5%/lvl |

### Island Exploration
16 named islands in order of `exploreProgress` threshold. Higher-tier islands require consuming a map of matching quality. Each grants one-time rewards and a permanent `BonusKey` boost.

Notable examples: Driftwood Cay (25 pts, free), Midnight Market (8000 pts, legendary map, +10% trade), World's End (11000 pts, legendary map, +12% treasure).

### Maps
4 tiers: Common / Rare / Epic / Legendary. Generated by Cartography/Tavern/islands. Consumed by: island discovery, Treasure Hunt (spent to dig). Higher tiers have better hunt success. Maps can be combined (2 commons → 1 rare, etc.).

### Treasure Hunting
Hunt Progress fills toward `HUNT_POINTS` (40); on completion, uses the best available map and rolls for treasure. Success chance scales with `relicFind` and `treasure` bonuses. Treasure is fenced for gold at port.

### Relics (12 items in 4 sets)
Found via Study Relics or island rewards. Each grants a permanent `BonusKey` bonus. Collecting all 3 in a set unlocks a set bonus.

| Set | Set Bonus | Items |
|---|---|---|
| Curse of the Deep | Relic Find +10% | Black Pearl Fragment, Kraken Tooth, Ghost Lantern |
| Captain's Regalia | XP +10% | Captain's Compass, Cursed Coin, Sea King's Crown |
| Navigator's Secrets | Treasure +10% | Star Chart, Brass Spyglass, Storm Glass |
| Smuggler's Cache | Trade +10% | Marked Doubloon, Shadow Flag, Hollow Barrel |

### Bosses (Bounty Board, 8 escalating)
Power ladder from 12 (Scurvy Dog Dan) to 40,000 (The Sea King). Each kill grants a permanent per-run bonus and gold/treasure rewards. Win chance = `fleetPower / (fleetPower + boss.power × 1.25)`. Bosses reset on prestige.

### Random Events (10 types, weighted)
Triggered every ~2–4 minutes. Present a choice modal. Examples:
- **Storm Front** — sail through (XP vs. loss) or shelter (safe, small cost)
- **Mutiny** — raise wages / make example / diplomacy check
- **Merchant Fleet** — attack (plunder) or trade
- **Kraken** — study / harvest tooth (relic chance) / flee
- **Ghost Ship** — board (cursed loot) or avoid
- Storm/Kraken events also trigger a visual storm effect in the 3D scene.

### Quests (5 parallel tracks)
Each track has sequential goals. Tracks: combat, wealth, exploration, fleet building, legend. Displayed in an always-visible goal box with hints and progress bars.

### Prestige — "Become a Legend"
**Requirement:** 25,000 reputation AND 2,500 influence (×4 per prestige). On prestige: reset resources, fleet, crew, buildings, maps, islands, skill levels. **Kept:** skill mastery, achievements, relics, legend points, prestige count. Earn 6–12 Legend Points on each prestige.

**Legend Upgrades (9 types, max 10 levels each):**
Tales of Glory (+10% XP), Golden Reputation (+10% gold), Favorable Winds (+5% speed), Treasure Sense (+10% treasure), Famous Captain (+10% recruit), Merchant Prince (+10% trade), Terror of the Seas (+10% combat), Cursed Affinity (+10% relic find), Master Shipwright (+10% build speed).

### Trade System
8 trade goods (Sugar → Exotic Goods) with dynamic prices that fluctuate and mean-revert. Trade routes (unlocked at Trading level 8, 16, 24…) provide passive income. Cargo held in the fleet can be sold at port.

### Offline Simulation
On load, the game simulates up to 12 hours of offline time using the same `tick(s, dt, { offline: true })` function (skips events/achievements for speed). Shows an offline summary modal on return.

### 3D Scene (Seven Seas Idle)
Cozy low-poly harbor. Features: animated ocean (vertex displacement), fleet ships that grow in count/tier with player progress, NPC traffic ships, lighthouse with point light, treasure pile on dock, particle foam, rain effect during storm events, day/night cycle, filmic tone mapping, soft shadows.

## Game 2 — Pirate Logistics

### Elevator Pitch
A Factorio-inspired real-time logistics simulation in a procedurally generated archipelago. Design production chains, assign shipping routes as "conveyor belts," expand your island empire, and fix bottlenecks before your economy grinds to a halt.

### Core Loop
```
Build production buildings on owned islands → assign ships to routes
→ ship resources between islands → unlock tech → send expeditions
→ discover + colonize new islands → expand production chains
→ build military for defense → research advanced tech → scale up
```

### Key Philosophy Differences from Seven Seas Idle
- **No offline progress** — the world only moves while the game is open.
- **Active management** — player designs the system; bottleneck reports surface exactly what's stuck.
- **Spatial** — islands exist at real coordinates; distance matters for route efficiency.
- **Simulation depth** — crew eat food and drink rum per-island; hungry islands work at half speed.

### Resources (15 total)
Raw: Wood, Stone, Iron Ore, Coal, Food, Sugar
Processed: Iron, Rum, Cloth, Cannons, Weapons, Ship Parts
Special: Maps, Treasure, Crew

### Island System
**Procedural generation:** ~27 islands generated from a seed using a deterministic PRNG (mulberry32). Home island at (0,0) always has Forest + Fertile + Stone. Guaranteed trait sets ensure all production chains are completable. Islands have x/z coordinates; distance drives travel time.

**Island traits (7 types):** Forest 🌲, Iron Deposit ⛏️, Coal Seam ⚫, Stone Quarry 🪨, Fertile Land 🌾, Ancient Ruins 🏛️, Treasure Site 💎

**Island state:** discovered (fog of war), owned, per-resource storage (capped by warehouses), list of buildings, looted flag (ruins/treasure sites loot once per wave).

**Storage:** base 80 per resource + 150 per warehouse level × industry tech multiplier. Crew housing: 12 base + 10/tavern level + 4/dock level.

### Buildings (15 types, max level 3)
Built from island local storage + gold. Build slots = 6 + floor(island.size × 4).

| Building | Trait Req | Tech Req | Key Recipe |
|---|---|---|---|
| Dock | — | — | Required for routes/expeditions; loads ships faster at higher level |
| Warehouse | — | — | +150 storage/level |
| Wood Camp | Forest | — | → 4 Wood / 6s |
| Mine | Iron/Coal/Stone | — | → 3 Iron Ore or Coal or Stone / 7s |
| Farm | Fertile | — | → 5 Food / 8s |
| Sugar Plantation | Fertile | — | → 3 Sugar /8s or 1 Cloth / 12s |
| Smelter | — | — | 3 Iron Ore + 1 Coal → 2 Iron / 8s |
| Rum Distillery | — | — | 3 Sugar → 2 Rum / 8s |
| Tavern | — | — | 1 Rum + 2 Food → 1 Crew / 12s |
| Shipyard | — | Shipbuilding 1 | 2 Iron + 3 Wood + 1 Cloth → 1 Ship Part / 10s |
| Fort | — | Military 1 | → Cannons or Weapons; defends vs. pirate raids |
| Marketplace | — | Trade 1 | Manual + auto buy/sell for gold |
| Treasure Vault | — | — | 1 Treasure → gold (special conversion) |
| Cartographer | — | Exploration 1 | Wood → Maps / 18s |
| Naval Academy | — | Military 2 | Rum + Weapons → influence (special) |

**Production simulation:** each building tracks recipe cycle progress. Buildings stall with explicit status codes: `noInput` / `full` / `noCrew`. Staffing scales output by `min(1, crew / workersNeeded)`. Level multiplier: `1 + (level-1) × 0.75`.

### Ships (6 types)
Cargo Ship → Sloop → Merchant Ship → Brig → Frigate → Galleon

Each has: capacity, speed (world units/s), maintenance cost (gold/min), crew requirement, build cost (gold + ship parts + cannons).

Ship states: `idle | route | expedition | colonize`. Route ships move through phases: `loading → toDest → unloading → toSource`. Condition degrades; storms damage hull.

### Shipping Routes
A route connects exactly one source island to one destination, carrying one resource type. Multiple ships can be assigned to the same route. Ships auto-cycle. Routes track: lifetime delivered, rolling rate (per minute window), utilization (avg cargo fraction).

### Tech Tree (7 branches, 3 levels each)
Costs: Gold + Maps + Influence at higher levels.

| Branch | Level 1 | Level 2 | Level 3 |
|---|---|---|---|
| Navigation | +15% ship speed, safer expeditions | +15% speed, much safer | +15% speed, nearly risk-free |
| Shipbuilding | Unlocks Shipyard + Sloop | Unlocks Brig + Frigate | Unlocks Galleon |
| Trade | Unlocks Marketplace + Merchant | +15% sell prices | +30% sell, cheaper imports |
| Smuggling | Rum+Weapons at 2× price (navy risk) | −15% maintenance | Navy rarely catches you |
| Military | Unlocks Fort | Unlocks Naval Academy | Forts repel nearly all attacks |
| Exploration | Unlocks Cartographer; +1 map/expedition | Expeditions loot more | 2× expedition speed |
| Industry | +20% production speed | +25% storage | +25% production speed |

### Expeditions
Cost: 3 crew + 10 food + 3 rum from the launching island. Requires idle ship + owned dock. Three kinds:
- **Explore:** charts nearest undiscovered island
- **Loot:** retrieves treasure/maps from ruins/treasure sites
- **Colonize:** costs 5 crew + 20 food + 40 wood + 20 stone; claims an island

### Economy / Market
Each marketplace island can trade. Prices fluctuate around base values (mean-reverting random walk). Market crash events halve sell prices / reduce buy prices for a duration. Auto-trade rules: sell excess above threshold / buy up to threshold automatically.

**Crew upkeep:** each island's crew consumes Food (0.012/crew/s) and Rum (0.006/crew/s) from local storage. Starvation → 0.5× efficiency. Sobriety → 0.75× of remaining. Combined: `(0.5 + 0.5 × fedRatio) × (0.75 + 0.25 × rumRatio)`.

**Smuggling:** rum + weapons at 2× price, but unlocking Smuggling tech 1 is required. Navy intercept chance varies by tech level (18% → 5%).

### Random Events (auto-resolve, no choices)
Mitigation is structural (build forts, research tech), not dialog-based:
- **Storm:** damages ships on active routes by 15–35 condition
- **Pirate Attack:** fort level determines defense; stolen goods if successful
- **Navy Raid:** smuggling players only; fined unless Smuggling 3
- **Treasure Discovery:** free treasure on ruins/treasure sites you own
- **Market Crash:** prices halved for a duration
- **Disease:** crew losses on crowded islands
- **Bonus Harvest:** double output on a random island
- **Naval Deserters:** free crew added to an island

Events fire every 90–180 s. Visual feedback: storms trigger 3D weather, pirate attacks pulse the island with a red flash, discoveries with gold.

### Production Statistics
Rolling rate tracker: 6 buckets × 10s = 60s window. Shows production/consumption per minute for every resource across the whole empire. Bottleneck report: lists every stalled building with island name, building name, and exact failure reason.

### 3D Scene (Pirate Logistics)
Top-down archipelago. Features: animated ocean, islands that grow visually with buildings (trait-colored terrain accents), ships sailing real routes in real time with smooth orientation, route lines on the ocean, fog of war (undiscovered islands hidden), raycaster click-to-select, hover tooltips, storm weather effects, pulse flash on events, camera pan-to-island on selection.

### UI Layout
Left half: full-width 3D map. Right half: side panel with 6 tabs.
- **Island** — selected island: traits, storage, buildings, build/upgrade/demolish, expeditions
- **Routes** — create/delete routes, assign/unassign ships, per-route stats
- **Ships** — fleet list, buy ships, condition, assign home island
- **Production** — empire-wide resource rates, bottleneck report
- **Tech** — 7-branch tree with costs and unlock status
- **Economy** — market prices, buy/sell, auto-trade rules, smuggling

Speed controls: ⏸ / 1× / 2× / 4×. Top bar shows: gold rate, island count, crew total, ship count.

## Design DNA — Similarities & Differences

| Dimension | Seven Seas Idle | Pirate Logistics |
|---|---|---|
| Pacing | Idle — set-and-forget | Active — continuous management |
| Offline | Up to 12 h simulated | No offline progress |
| Skill expression | Which activity to focus on; upgrade order | Production chain design; route architecture |
| Goal structure | Explicit quest tracks + prestige layers | Emergent from tech tree + empire growth |
| Events | Choice modals — narrative, consequential | Auto-resolve — mitigation is structural |
| 3D scene | Cozy harbor that grows with progress | Live map of moving ships and islands |
| Reset mechanic | Prestige with mastery/legend carry-over | None — one long run |
| Tone | Roleplaying as a captain | Systems/logistics management |

**Shared design values:**
- Pure TypeScript; no framework overhead; innerHTML re-render on a timer
- All rates are dt-robust (same formula works for 1 tick or offline chunks)
- Numbers abbreviated via shared `fmt()` utility
- Local save every 5 s, export/import JSON
- Emoji icons throughout (no external asset dependencies)
- Three.js primitives-only; no models; flat shading

## Known Design Opportunities (for future sessions)

1. **Seven Seas Idle prestige depth** — only one prestige currency; a second axis (e.g. "Infamy" for combat players) could branch run styles.
2. **Pirate Logistics endgame** — no current "win state"; a naval campaign or faction war could give a late-game goal.
3. **Cross-game meta** — the landing page exists; a shared "pirate legacy" score could link the two games.
4. **Mobile layout** — both games currently assume landscape desktop; the 3D panel vs. side-panel split doesn't adapt to small screens.
5. **Seven Seas sound** — no audio; ambient ocean/crew sounds would reinforce the cozy tone with minimal asset weight (Web Audio API).
6. **Pirate Logistics trade routes quality-of-life** — no route grouping or bulk-assign; scaling to 15+ islands with 40+ routes becomes tedious.
7. **Achievements in Pirate Logistics** — the game has none; Seven Seas Idle's achievement system could be ported across.
